Lesson 1 - The First Tag




The first thing is about to understand HTML's base structure. Watch the image shown below, the html tag contain the head and body tag. The head tag link css and js files, sets the document title, sets the viewport, etc. The head tag's content never show in the document. But the title show as the tab title near the close tab button. The body tag contain the all componements, which are output to the viewer. Any code inside body tag will output.

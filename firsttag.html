Lesson 1 - The First Tag




The first thing we should understand is about HTML's base structure. View the image shown below, the html tag contains the head and body tag. The head tag links css and js files, sets the document title, viewport, etc. The head tag's content never appears in the document. But the title appears in the tab title near the close tab button. The body tag contains all the components, which is the output view to the viewer. Any code inside body tag will become the output.

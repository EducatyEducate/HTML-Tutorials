Lesson 3 - Headings




Like in sections of a website, there are several headings. In heading tags, it is possible to add a long text, But it is recomened to use the heading tag only for like a title to show in the document. Watch the given image, it show the heading tags. There are total 6 heading tags, the h1 (first heading) tag is the biggest, h2 is a bit small, h3 is some more small and so on! The h6 as the last heading tag is the smallest heading, which is also smaller than a paragraph.

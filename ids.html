Lesson 3 - Ids




Ids are used to identify the element. Ids are represented using the # (hash) symbol. When we create a element and set a id attribute as a value, we can identify it with #id. We can use the #id method in css to customize the display of all the elements which have '#id' id set. The same look will be for all the elements which have the '#id' id. Its most commonly used in JS. In JavaScript, we can identify the element using js to make it functional.
